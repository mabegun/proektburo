## 🎯 Использование с ИИ

Эта документация оптимизирована для работы с ИИ-ассистентами (ChatGPT, Claude, Qwen Code).

### Как использовать:

1. **Скопируйте содержимое** нужного файла
2. **Вставьте в чат** с ИИ
3. **Добавьте промпт:**
   ```
   Изучи эту документацию и создай backend/main.py 
   с FastAPI приложением и всеми endpoints из 03_API_ENDPOINTS.md
   ```

### Примеры промптов:

**Для создания БД:**
```
Используя 02_DATABASE_SCHEMA.md, создай backend/database.py 
с функциями init_db() и get_db_connection()
```

**Для создания API:**
```
Используя 03_API_ENDPOINTS.md, создай backend/main.py 
с роутами для проектов (GET, POST, PUT, DELETE)
```

**Для CRUD:**
```
Используя 02_DATABASE_SCHEMA.md и 03_API_ENDPOINTS.md, 
создай backend/crud.py с функциями для работы с проектами
```

**Для авторизации:**
```
Используя 04_ACCESS_CONTROL.md, создай backend/auth.py 
с проверкой JWT токенов и прав доступа
```
