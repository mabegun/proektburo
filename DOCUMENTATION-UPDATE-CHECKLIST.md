## 📍 Где настраиваются ссылки

### Навигация между страницами:

| Файл | Где править ссылки |
|------|-------------------|
| Сайдбар | Каждый HTML файл, блок `<aside>` |
| 00-map.html | Блоки с `<a href="...">` |
| 13-settings.html | Блок "Справочники" |
| Хлебные крошки | Каждый HTML файл, блок breadcrumb |
| Вкладки справочников | 13-1-settings-sections.html, блок tab-item |

### Перекрёстные ссылки:

| Страница | Ссылается на |
|----------|--------------|
| 03-projects-list.html | → 04-project-detail.html, 05-project-create.html |
| 04-project-detail.html | → 06-section-detail.html, создание раздела |
| 06-section-detail.html | → 04-project-detail.html, задачи |
| 07-tasks-list.html | → 08-task-create.html, детали задачи |
| 09-employees-list.html | → 10-employee-create.html |
| 13-settings.html | → 13-1, 13-2, 13-3 |
| 13-1-settings-sections.html | → 13-2, 13-3 (вкладки) |

## 🔄 Порядок коммитов

```bash
# 1. Добавить/изменить HTML файлы
git add *.html

# 2. Добавить документацию
git add *.md

# 3. Коммит с описанием
git commit -m "feat: добавить страницу X"

# 4. Push в репозиторий
git push origin main
```
