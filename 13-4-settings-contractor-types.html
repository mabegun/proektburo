Сотрудник — Штатный работник

Активен

Штатный сотрудник организации. Выплата через зарплату/аванс, документы не требуются.

Обязательные документы

Не требуются

Примечание

Выплата производится через бухгалтерию в рамках трудового договора.

О типах контрагентов

Тип контрагента определяет обязательный набор документов для выплат. При создании выплаты тип подставляется автоматически из профиля исполнителя, но может быть изменён. Система будет подсказывать, какие документы нужно загрузить для получения выплаты.
